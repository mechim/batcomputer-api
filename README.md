# Batcomputer API
Inspired by [Wizard World API](https://github.com/MossPiglets/WizardWorldAPI), this is a simple REST API for batman to manage his detective work. It is very simple and created purely for practice with Spring Boot.

## Getting Started
### Dependencies 
 * If you want to test out the API you only need `Docker`.

### Instructions
 * Download the `docker-compose.yml` file
 * Create a `.env` file as per the example in `.env.example`
 * Run

   ```
   
   docker-compose up
   
   ```
 * Swagger documentation is available with this link
    ```
    
     http://localhost:8080/swagger-ui/index.html#/
    
    ```
